# ilo4_exporter
Metrics exporter for HP iLO4 to prometheus

## Install
```
go get -u github.com/MauveSoftware/ilo4_exporter
```

## Usage
### Start server
```
./ilo4_exporter -api.username=admin -api.password=password
```
